==================
Terraform Module Collection
==================

For Easy Provide Infrastructure, here some Terraform modules. The Modules wrappped the `hcloud <https://www.terraform.io/docs/providers/hcloud/index.html>`_ terraform provider, with some preconfiguration elements.
This modules build the basic Infrastructure, from `nolte/personal-storage-infrastructure <https://github.com/nolte/personal-storage-infrastructure>`_ and `nolte/minecraft-infrastructure <https://nolte.github.io/minecraft-infrastructure/index.html>`_.
